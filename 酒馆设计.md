# 0. 酒馆底层架构
卡牌模板用于描述卡牌，卡牌实例在游戏开始前与游戏内可以随时创建与销毁，并且卡牌实例中的特效是可以添加删除与修改的

玩家角色同样是一种特殊的卡牌实例，其等级为玩家的酒馆等级，当前晶体矿，当前晶体矿上限，晶体矿最大上限，当前瓦斯与升本需要消耗的晶体矿作为特殊单位在单位数组中记录

特效模板用于描述特效，监听到对应event时便会尝试判断condition是否满足并尝试消耗cost来执行operation。使用slot进行状态记录，不同的特效实例可以关联到同一个slot用于维护特效状态。

# 1. struct CardTemplate 
int templateId
- 模板ID，在游戏开始前就被创建，通常与所在卡牌模板数组下标相同

string cardName
- 卡牌名称

int level
- 卡牌等级

int faction
- 卡牌阵营（TPZ中立）

int subfaction
- 卡牌子阵营（泰伦人族，原始异虫，塔达林等等）

int type
- 卡牌类型，决定卡牌是原始卡牌，三连卡牌，被三连卡牌，虚空捏的牌等等

bool canMerge
- 卡牌能否三连

int unitIds[211]
- 卡牌内单位的id，1-200为通常单位，201-210为特殊单位（科技挂件，水晶塔等）

int unitAmounts[211]
- 对应单位的数量

int effectIds[17]
- 特效实例id表

# 2. struct EffectTemplate

特效模板在游戏开始前被创建只描述“这个效果应该如何工作”，不记录具体卡牌当前状态。

int effectTemplateId
- 模板ID，在游戏开始前由数据定义，通常与所在特效模板数组下标相同

int effectTypeId
- 效果类型，主动效果与诱发效果，用于关联UI是按钮还是进度图标

int listenEventId
- 监听时点id

int conditionFuncId
- 发动条件判断函数id

bool costFuncId
- cost支付函数id

int targetSeletFuncId
- 效果目标数组选择id

int operationFuncId
- 效果函数引用id

int linkedEffectTemplateID[8]
- 关联效果的id，在效果实例化时自动将被记录的效果一并实例化并登录到同一个状态slot里

string paramArray[8]
- 额外参数表

# 3. struct CardInstance

在游戏初始化及游戏中，可以创建，删除，修改卡牌实例并与之交互

int templateId
- 模板ID，在游戏开始前就被创建，通常与所在卡牌模板数组下标相同

int instanceId
- 实例ID，在游戏初始化时及游戏时被创建

string cardName
- 卡牌名称

int level
- 卡牌等级

int faction
- 卡牌阵营（TPZ中立）

int subfaction
- 卡牌子阵营（泰伦人族，原始异虫，塔达林等等）

int type
- 卡牌类型，决定卡牌是原始卡牌，三连卡牌，被三连卡牌，虚空捏的牌等等

bool canMerge
- 卡牌能否三连

bool isTemporary
- 是否是临时创建的卡牌，虚空捏的卡不会返回卡池

bool isMerged
- 卡牌是否被三连合并到某个卡牌里

int mergedCardIds[4]
- 被合并到这个卡牌里的实例id

int unitIds[211]
- 卡牌内单位的id，1-200为通常单位，201-210为特殊单位（科技挂件，水晶塔等）

int unitAmounts[211]
- 对应单位的数量

int effectIds[17]
- 特效实例id表

int slotIds[17]
- 特效实例对应的slot

int ownerPlayerId

int slotInstanceId

int effectTypeId
- 效果类型，主动效果与诱发效果，用于关联UI是按钮还是进度图标

int listenEventId
- 监听时点id

int conditionFuncId
- 发动条件判断函数id

bool costFuncId
- cost支付函数id

int targetSeletFuncId
- 效果目标数组选择id

int operationFuncId
- 效果函数引用id

string paramArray[8]
- 额外参数表

# 5. struct SlotInstance

slot实例,记录某个或某些特效的状态参数。

int ownerCardInstanceId
- 所属卡牌实例的id

int ownerPlayerId
- 所属玩家id

int slotParameters[8]
- 记录数值型的状态参数

string slotParameters[8]
- 记录字符型的状态参数
