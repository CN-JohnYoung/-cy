# 1. 全局变量
全局变量的命名应当具有前缀，其中全局变量为gv_，全局函数为gf_，全局结构体应为gs_，全局常量应为gc_。

对于函数参数，前缀为lp_。对于结构体与函数体内的局部变量，前缀为lv_。

# 2. 数组声明
对于galaxy，数组长度也是数据类型的一员，通常来说数组下标0不会正式使用，可用作边界判定等用途，GUI在生成galaxy代码时，会自动在数组长度后面+1，且数组长度应放到类型后面，如int[length + 1] a;

通常情况下，直接生成的galaxy语言应当避免直接定义数组长度，如int[8] a，应当将数组长度定义为一个全局常量，并且在声明数组时依照GUI生成galaxy代码的规则，自动在后面+1。
